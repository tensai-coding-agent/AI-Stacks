# AI Stacks

A monorepo for all AI side projects.

## Structure

```
├── packages/        # Shared libraries and components
├── projects/        # Individual AI applications
├── tools/           # Development and build tools
└── docs/            # Documentation
```

## Getting Started

Each project in `projects/` has its own README with setup instructions.

## Contributing

1. Create a feature branch: `git checkout -b feature/PROJECT-XX-description`
2. Commit changes with clear messages
3. Open a Pull Request for review
