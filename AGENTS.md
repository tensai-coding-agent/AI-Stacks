# AI Stacks - Agent Instructions

## Overview

This is a monorepo containing AI side projects. When working in this repository:

## Project Structure

- `packages/` - Shared libraries usable across projects
- `projects/` - Individual AI applications (each with its own setup)
- `tools/` - Build scripts, dev tools
- `docs/` - Shared documentation

## Development Guidelines

1. **Branch Naming**: `feature/{TICKET-ID}-{brief-description}`
2. **Commits**: All commits must end with `Co-Authored-By: Paperclip <noreply@paperclip.ing>`
3. **No Direct Main Pushes**: Always use Pull Requests

## Adding New Projects

When creating a new AI project:
1. Create folder under `projects/`
2. Include its own README with setup instructions
3. Add to root README's project index
